# Kafka Desktop Viewer

A lightweight desktop app for tailing Kafka topics locally (Docker Compose) with a safe UI:
- Topic listing + start/stop consuming
- Earliest/latest
- Search + message details
- Masking enabled by default
- Bounded memory (max 500 messages)

## Download (recommended)
Go to the GitHub Releases page and download:
- macOS: `.dmg`
- Windows: `.exe`
- Linux: `.AppImage`
